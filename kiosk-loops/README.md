# Kiosk Ticket Processor

A simple Java console program that demonstrates processing kiosk ticket numbers using both `while`-loop and `for`-loop approaches.

## Features

- Processes a user-defined number of tickets
- Demonstrates both `while`-loop and `for`-loop implementations
- Proper variable scope and loop control
- Clean, commented code with correct terminology

## How to Compile

```bash
cd kiosk-loops/src
javac KioskTicketProcessor.java
```

## How to Run

```bash
java KioskTicketProcessor
```

## Example Usage

```
How many tickets are in today's queue? 3

--- Part 1: Processing with while-loop ---
Enter next ticket number: T001
Processed ticket #T001
Enter next ticket number: T002
Processed ticket #T002
Enter next ticket number: T003
Processed ticket #T003

--- Part 2: Processing with for-loop ---

All tickets processed. Have a great day!
```
