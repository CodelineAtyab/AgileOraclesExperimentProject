# Weather Forecast API with Postman

## Successful Request
Method: GET

URL:
https://api.open-meteo.com/v1/forecast?latitude=23.5880&longitude=58.3829&current_weather=true&timezone=Asia/Muscat

Header:
Content-Type: application/json

## Successful Response
Status: 200 OK

Observation:
The API returned current weather data for Oman in JSON format. The response included weather fields such as time, temperature, wind speed, wind direction, and weather code.

## Response Anatomy
Status code: 200 OK
Headers observed: Date, Content-Type, Connection
Body: JSON weather data

Observation:
The HTTP response contains a status code, headers, and body. The status code shows success or failure, headers provide metadata, and the body contains the actual data.

## Negative Test
Changed latitude to: abc

Result:
400 Bad Request

Observation:
The API rejected the invalid latitude value because latitude must be numeric. This shows that the API validates query parameters.
